# Crystal Mathematic

The objective of this project is to create a mathematical library coded 100% in Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

  ```yaml
  dependencies:
    crystal_mathematic:
      github: lodenos/Crystal_Mathematic
  ```

2. Run `shards install`
